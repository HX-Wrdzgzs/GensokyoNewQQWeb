# WebUI 管理面板

Gensokyo-NewQQ 提供基于 Web 的管理界面，方便你实时监控和管理机器人实例。

## 访问地址

启动 Gensokyo 后，在浏览器中访问控制台地址（默认为 `http://<你的IP>:<端口>/webui/`）。

初始用户名和密码为 `admin` / `admin`，请及时修改。

## 界面概览

WebUI 采用现代化暗色优先设计，整体布局分为：

### 左侧导航栏

- **仪表盘** — 系统运行状态总览
- **添加机器人** — 创建新的机器人实例
- **机器人列表** — 所有已配置的机器人

### 仪表盘页面

仪表盘展示以下关键指标：

- **CPU 占用** — 系统和主进程 CPU 使用率
- **内存占用** — 系统和主进程内存使用情况
- **硬盘占用** — 磁盘空间使用率
- **进程状态** — 主进程内存、开机时间等信息
- **资源趋势图** — 实时更新的 CPU/内存使用趋势曲线
- **进程日志** — 实时日志流，支持 ANSI 彩色输出

### 机器人详情页

点击机器人列表中的某个机器人进入详情页：

- **进程状态** — 查看运行状态、CPU、PID、内存、运行时间
- **启动/停止** — 控制机器人进程
- **快速发消息** — 向群聊或好友发送测试消息
- **实时日志** — 终端风格的日志输出
- **配置编辑** — 在线编辑 `config.yml`
- **设备信息** — 编辑设备信息，支持 QDVC 导入导出

### 频道/群列表

查看和管理机器人加入的所有频道和群组，支持：

- 分页浏览
- 多选后批量发送消息
- 点击进入子频道详情

## 设计主题

WebUI 支持亮色/暗色主题切换，默认跟随系统偏好。暗色模式采用深色背景配合渐变强调色，提供舒适的夜间使用体验。
