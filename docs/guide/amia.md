# Amia — 晓山瑞希

基于 Gensokyo 生态构建的示例机器人项目，展示了如何利用 Gensokyo 的 OneBot V11 协议网关快速搭建功能完整的 QQ 机器人。

## 项目简介

**Amia（晓山瑞希）** 是一个开源的 QQ 机器人示例项目，集成了多种实用功能模块，可作为开发参考或直接部署使用。

- **技术栈**: Python + NoneBot2 / Go + Gensokyo
- **协议层**: 基于 Gensokyo 的 OneBot V11 标准协议
- **功能覆盖**: 群管、娱乐、实用工具、多媒体处理

## 功能特性

- 🤖 **完整的 OneBot V11 支持** — 利用 Gensokyo 的 WebSocket/HTTP 接口
- 📦 **模块化插件系统** — 方便扩展和定制化开发
- 🎨 **丰富的交互能力** — 支持 Markdown 消息、按钮交互、多媒体发送
- 📊 **内置监控与日志** — 搭配 Gensokyo WebUI 实时查看运行状态

## 快速开始

参考 Gensokyo 的配置方式，将 Amia 连接到你的 Gensokyo 实例：

1. 确保 Gensokyo 已正确配置并运行
2. 配置 `ws_address` 指向你的应用端地址
3. 启动 Amia，即可通过 OneBot 协议与 QQ 平台交互
