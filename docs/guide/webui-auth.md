# 登录与账户管理

## 登录

首次访问 WebUI 时，会显示登录页面。输入用户名和密码进行身份验证。

**默认凭据**:
- 用户名: `admin`
- 密码: `admin`

> 请在生产环境中及时修改默认密码。

## 账户管理

### 添加机器人

1. 在左侧导航栏点击「添加机器人」
2. 输入机器人的 AppID
3. 点击「提交」创建新实例

### 管理机器人

在左侧导航栏的机器人列表中，点击对应机器人可进入详情页：

- **查看状态** — CPU、内存、运行时间
- **启动/停止** — 控制进程启停
- **编辑配置** — 在线修改 `config.yml`
- **编辑设备** — 修改设备信息文件
- **发送消息** — 快速向群聊/好友发送测试消息

### 删除机器人

在机器人列表中，预定义机器人（来自配置文件）不可删除，手动添加的机器人可以删除。
